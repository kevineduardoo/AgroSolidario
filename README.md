# 🌱 AgroSolidario

Sistema de Gestão de Doações de Alimentos desenvolvido com ASP.NET Core MVC.

## 📋 Sobre o Projeto

O AgroSolidario é uma plataforma que facilita a gestão de doações de alimentos, conectando doadores a beneficiários como escolas, ONGs, hospitais e famílias carentes.

## ✅ Funcionalidades

- 🔐 Login, Logout e Cadastro de usuários
- 👥 Controle de acesso por tipo (ADMIN, Doador, Beneficiário)
- 📦 Cadastro de alimentos com foto
- 🔍 Pesquisa de alimentos
- ✅ Marcar alimento como Entregue/Pendente
- 📊 Dashboard com gráficos e estatísticas
- ⚠️ Notificação de alimentos próximos da validade
- 📋 Histórico de doações
- 📄 Relatório em PDF
- 👤 Página de perfil do usuário
- ℹ️ Página sobre o projeto

## 🛠️ Tecnologias

- ASP.NET Core MVC
- Entity Framework Core
- SQLite
- Bootstrap 5
- Bootstrap Icons
- Chart.js
- QuestPDF

## 🚀 Como Rodar

1. Clone o repositório:
```bash
git clone https://github.com/kevineduardoo/AgroSolidario.git
```

2. Entre na pasta do projeto:
```bash
cd AgroSolidario/AgroSolidario
```

3. Execute:
```bash
dotnet run
```

4. Acesse no navegador:
5. http://localhost:5283

   > O banco de dados SQLite será criado automaticamente na primeira execução.
